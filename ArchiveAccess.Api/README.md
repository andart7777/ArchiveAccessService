# Archive Access Service

Служба доступа к данным электронного архива документов. (D_86124 диплом / ВКР Артем )

## Требования

На компьютере должен быть установлен Docker Desktop или Docker Engine с Docker Compose.

## Запуск

1. Распаковать архив.
2. Перейти в папку проекта:

```bash
cd ArchiveAccess.Api
